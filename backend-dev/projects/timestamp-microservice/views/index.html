Timestamp REST API

This is a simple REST API that returns a JSON object with a unix timestamp and a natural language date.

The API accepts a date parameter in the form of a unix timestamp or a natural language date (e.g. January 1, 2016).

If the date parameter is invalid, the API returns a JSON object with a null value for both properties.

Usage

GET /api/2022-06-16

GET /api/06-16-2022

GET /api/1655385600

{
  "epoch": 1655385600,
  "natural": "June 16, 2022"
}
